# Cleaning Platform

Multi-tenant cleaning services platform (React + Node.js + Supabase).

## Local development

### Backend
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your database and API keys
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

Frontend: http://localhost:3000  
Backend API: http://localhost:5000
